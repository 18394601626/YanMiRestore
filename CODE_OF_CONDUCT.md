# 社区行为准则

我们致力于为所有参与者提供一个专业、友善、可协作的社区环境。

## 我们鼓励的行为

- 以事实和证据进行技术讨论
- 尊重不同经验背景与观点
- 接受建设性反馈，并愿意修正错误
- 聚焦问题本身，避免人身攻击
- 在公开讨论中避免泄露敏感信息

## 不可接受的行为

- 侮辱、骚扰、威胁、歧视性言论
- 恶意引战、刷屏、持续性人身攻击
- 未经授权公开他人隐私或敏感数据
- 发布违法用途指导（含未授权取证与入侵内容）
- 明知违规仍反复破坏社区秩序

## 适用范围

本准则适用于项目相关的公开沟通场景，包括但不限于：

- Issue / Pull Request / Discussion
- Commit 评论与评审讨论
- 项目维护者组织的线上协作渠道

## 违规处理

维护者可根据情节采取以下措施：

1. 口头提醒或书面警告
2. 删除不当内容
3. 暂时限制互动权限
4. 永久封禁参与权限

## 举报方式

如遇违规行为，请通过仓库维护渠道联系维护者，并提供：

- 事件链接
- 发生时间
- 简要经过
- 必要截图（可选）

维护者会在合理时间内处理，并尽量保护举报者隐私。
